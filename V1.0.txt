Календарь v1.0 - Нет "Статистики" (создана кнопка но без функционала) и остались темы жидкого стекла

---
### 📊 Расписание
- Плоский список через `ScheduleBuilder`
- Заголовки месяцев
- Дни с событиями (плашки с цветной линией)
- Свободные диапазоны (без дублирования, не пересекают месяцы)
- Клик по событию → экран деталей
- Свайп для удаления с подтверждением
---
🎯 Виджет «День» — детали
- Дата: день недели (красный) + число (белый)
- До 4 событий: цветная точка + время + название
- Выполненные задачи — зачёркнуты
- Кнопка "+" → создание события
- Клик по событию → экран редактирования
- Полночь → обновление даты через AlarmManager

PDF экспорт:
- Полноценная поддержка кириллицы (Roboto шрифт)  - удалить всё что связано с экспортом PDF.
- Дата, время, категория, место, описание
Импорт праздников:
- Nager.Date API (бесплатно, 100+ стран)
- Автоопределение страны по локали
- Автозагрузка при старте + при смене года

### 📍 Место проведения
- **Поиск места** через Nominatim API (OpenStreetMap)
- Автодополнение при вводе (debounce 1000мс)
- Кнопка «Моё местоположение» — обратный геокодинг
- Сохранение: название + адрес + координаты
- Открытие в картах (Google Maps, Яндекс, 2ГИС)
- Экспорт координат как ссылка `https://maps.google.com/?q=...`
